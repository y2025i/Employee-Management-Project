Employee Management System 🏢

This project is developed for the Advanced Programming lecture. It is a desktop-based Employee Management System built using Python's Tkinter library for the Graphical User Interface (GUI) and MySQL for database management.

🌟 Features
Add Employee: Insert new employee records into the database.

Update Employee: Modify existing employee details.

Delete Employee: Remove an employee from the system.

Search Employee: Find specific employees by their ID or Name.

Clear Form: Easily clear input fields for new data entry.

Display All: View all employee records in a tabular format.

🛠️ Technologies Used
Programming Language: Python 3.x

GUI Framework: Tkinter

Database: MySQL

Database Connector: pymysql

🚀 Installation & Setup
Prerequisites
Python 3.x installed

MySQL Server installed and running

pymysql library installed (pip install pymysql)

Database Setup
Open your MySQL Workbench or terminal.

Create a new database (e.g., employee_db).

Create the required employee table (you can run the provided .sql file if available).

Update the database credentials (username, password) in the Python connection script.

Running the App
Clone this repository:

bash
git clone [Your-GitHub-Repository-URL]
Navigate to the project folder.

Run the main Python file:

bash
python main.py
